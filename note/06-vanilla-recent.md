## 최근 검색어 구현 by VanillaJS
추천 검색어 탭 클릭 시에 하단에 나오는 검색어 목록

#### 요구사항
[ ] 최근 검색어, 목록이 탭 아래 위치한다   
[ ] 목록에서 검색어를 클릭하면 선택된 검색어로 검색 결과 화면으로 이동   
[ ] 검색일자, 버튼 목록이 있다    
[ ] 목록에서 x 버튼을 클릭하면 선택된 검색어가
